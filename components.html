Notifications

Example

  • Create notification board
    const notificationBoard = new Notification({
	delay: 40000
});
  • Create and show notification message
    const button = document.querySelector("button");
button.addEventListener("click", () => {
	notificationBoard.show("Short test message", {
		className: "notification_type_info"
	});
});

Options

Notification
  • delay
    Delay for closing event in milliseconds (ms).
    • Type: number
    • Default: 3000
  • containerClassName
    Notification board container class name
    • Type: string
    • Default: "notification-container"
  • containerItemClassName
    Notification board container item class name
    • Type: string
    • Default: "notification-container__item"
NotificationItem
  • delay
    Delay for closing event in milliseconds (ms).
    • Type: number
    • Default: 3000
  • close
    Show close button when true
    • Type: bollean
    • Default: true
  • className
    Custom class/classes name added to notification item
    • Type: string
    • Default: ""
  • containerClassName
    Notificiation item container class name
    • Type: string
    • Default: "notification"
  • disappearClassName
    Class name added when notification disappear
    • Type: string
    • Default: "notification_disappear"
  • closeClassName
    Class name to close button
    • Type: string
    • Default: "notification__close"
  • messageClassName
    Class name added to message
    • Type: string
    • Default: "notification__message"
